Fluxo simples

Como funciona na prática

Tudo foi pensado para reduzir atrito e deixar a organização clara para Admin e jogadores.

1

Admin cadastra os jogadores

O Admin adiciona cada jogador com nota e posição inicial. A senha inicial é usada no primeiro acesso.

2

Jogador entra por nome + senha

O atleta acessa com o nome cadastrado e sua senha, podendo atualizar os dados na própria tela.

3

Confirma presença e escolhe time

O jogador confirma presença, marca ausência se necessário e escolhe o time dentro das vagas disponíveis.

4

Admin compartilha a rodada

O Admin pode copiar tabela, baixar lista, mini lista, salvar no histórico e compartilhar no WhatsApp.
